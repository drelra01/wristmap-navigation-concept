# WristMap Navigation Concept

## Overview

WristMap is a wearable navigation system concept designed to help users travel with GPS routing, real-time traffic updates, touchscreen controls, voice commands, and extended battery life. The goal of the project was to design a standalone navigation device that could provide map and route information without depending on a smartphone.

## My Contribution

This was a team project, and my contributions focused on system design, implementation planning, risk analysis, and testing documentation. I worked on the front-end and back-end architecture for the hardware requirement, helped prepare the software architecture and risk analysis sections, and contributed to the software implementation plan. I also developed parts of the testing strategy, including functional, performance, regression, alpha, and beta testing, and helped create test reports and data visualizations for the final presentation.
